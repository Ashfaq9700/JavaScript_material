JavaScript

JavaScript (js) is a light-weight object-oriented programming language which is used by several websites for scripting the webpages. It is an interpreted, full-fledged programming language that enables dynamic interactivity on websites when applied to an HTML document. It was introduced in the year 1995 for adding programs to the webpages in the Netscape navigato browser.Since then, it has been adopted by all other graphical web browsers. With JavaScript, users can build modern web applications to interact directly without reloading the page every time.

External JavaScript

External JavaScript are practical when the same code is used in many different web pages. JavaScript files have the file extention .js
To use an external script, put the name of the script file in the src attribute of <script>tag. Ex: <script src="hello.js">
External javascript files can be palced in head and body section of HTML and there is no limit of linking external javascript files inside the HTML body or head.

variable is a "named storage" for data. we can use vaibales to store the information.
To create variable in JavaScript use the let keyword.
The statment below creates (in other words:declares) a variable with the name "message":
let message;
Now, we can put some data into it by using the assignment operator =:
message = "Welcome to JavaScript";
The string is now saved into the memory area associated with the variable.We can access it using the variable name:
alert(message);
